> The best module ever.


## Install

```sh
$ npm install --save rxflux
```


## Usage

```js
var rxflux = require('rxflux');

rxflux('Rainbow');
```

```sh
$ npm install --global rxflux
$ rxflux --help
```

```sh
# creates a browser.js
$ npm run browser
```
